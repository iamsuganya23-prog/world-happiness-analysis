# 🌍 World Happiness Report Analysis (2015-2019)

## Overview
Exploratory data analysis of the World Happiness Report dataset using Python and pandas.
Data covers 150+ countries across 5 years (2015-2019).

## Tools Used
- Python
- pandas
- matplotlib

## What I Analyzed
- Top 10 happiest countries in 2019
- World happiness trend from 2015 to 2019
- Top 5 happiest countries (5 year average)
- Relationship between GDP and happiness score

## Key Findings
- Finland is the happiest country in 2019
- Nordic countries dominate the top 5 consistently
- Higher GDP per capita strongly correlates with higher happiness scores
- World average happiness score slightly increased from 2015 to 2019
